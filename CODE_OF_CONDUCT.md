# Code of Conduct

We pledge to make participation harassment-free for everyone.

## Standards
- Welcoming, inclusive language
- Respectful of differing viewpoints
- No personal attacks, trolling, or harassment

## Enforcement
Report issues to [@LIN4CRE](https://github.com/LIN4CRE).

Adapted from [Contributor Covenant 2.1](https://www.contributor-covenant.org).
